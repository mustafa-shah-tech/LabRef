# LabRef

Medical Lab Test Reference Ranges - A sleek, mobile-first web application for quickly referencing common medical laboratory test ranges, including clinical indications for high and low results.

![LabRef Screenshot Placeholder](#)

## Features
- **Instant Search**: Search test names and filter in real-time as you type.
- **Category Filtering**: Quickly filter tests by Hematology, Biochemistry, Liver, Kidney, or Thyroid.
- **Dual Themes**: Beautifully crafted dark (default) and light themes with automatic state saving via LocalStorage.
- **Glassmorphism Design**: Modern aesthetic using precise blur effects and transparent layers.
- **Responsive Layout**: Designed mobile-first, scales perfectly across phones, tablets, and desktops using a dynamic grid.
- **Offline Capable**: Zero external dependencies (apart from Google Fonts), everything runs locally in the browser with no server needed.

## Tech Stack
- Vanilla HTML5
- Vanilla CSS3 (Custom Properties, Flexbox, CSS Grid)
- Vanilla JavaScript (ES6+, DOM Manipulation)

## How to Use
This project requires no build tools, package managers, or server environment. Simply download the files and double-click `index.html` to open it in any modern web browser and start using the app instantly.

> **Disclaimer**: Data provided in this application is for general reference only. Laboratory reference ranges can vary between different labs and methods. Always consult a qualified medical professional for health concerns and interpretation of any tests.
